QUICK START GUIDE

=== BEFORE RUNNING ===

1. Ensure MySQL is running
   - On macOS: brew services start mysql

2. Create the database
   - Open MySQL client: mysql -u root -p
   - Run the script: source Database/script.sql

3. Update appsettings.json with your MySQL password (if any)

=== RUNNING THE PROJECT ===

1. Navigate to project directory:
   cd "/Users/sagarjaiswal/Assignment 2/AppointmentBooking"

2. Restore dependencies (first time only):
   dotnet restore

3. Build project (first time only):
   dotnet build

4. Run the project:
   dotnet run

5. Open browser:
   https://localhost:7069/swagger/index.html

=== FIRST API CALL - REGISTER USER ===

POST https://localhost:7069/api/auth/register
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "Password123"
}

Response includes JWT token - copy it.

=== SUBSEQUENT API CALLS ===

Include Authorization header:
Authorization: Bearer {your_jwt_token}

=== EXAMPLE WORKFLOW ===

1. Register user: POST /api/auth/register
2. View doctors: GET /api/doctors
3. Get available slots: GET /api/slots/available
4. Book appointment: POST /api/appointments
5. View my appointments: GET /api/appointments/my-appointments

=== ADMIN OPERATIONS ===

1. Login with admin user (create one first)
2. Create doctor: POST /api/doctors (with admin token)
3. Create slots: POST /api/slots (with admin token)

=== TESTING WITH POSTMAN ===

1. Import: Postman_Collection.json
2. Set variable {{token}} = JWT token from login
3. Test all endpoints

=== TROUBLESHOOTING ===

- Check MySQL is running
- Verify connection string in appsettings.json
- Check database exists: SHOW DATABASES;
- Check tables: USE appointmentbooking; SHOW TABLES;
- View logs in console output
